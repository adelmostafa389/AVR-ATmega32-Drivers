# AVR-ATmega32-Drivers
This Library contains Drivers for ATmega32 AVR Microcontroller with layered Architecture principle applied.  
## Drivers list  
**MCAL Layer:**  
  * ADC driver  
  * DIO driver  
  * Interrupt  
  * Timer0  
  * UART  
  
**ECUAL Layer:**
  * Button  
  * Character LCD 2x16  
  * Keypad  
  * LED
